Node.js: Professionell hochperformante Software entwickeln

Download this project as a .zip file Download this project as a tar.gz file

Mit Node.js wurde eine auf JavaScript und Google V8 basierende Ausführungsplattform für kommunikationsintensive Anwendungen entwickelt, die ihre Wurzeln zwar im Wesentlichen in der Web 2.0 Ära hat, sich aber immer weiter verbreitet. Das Buch soll zeigen, wie unter Node.js in einem professionellen Entwicklungsumfeld hochperformante Software geschrieben wird - dem Spannungsfeld zwischen JavaScript und Enterprise Software zum Trotz. Anhand der Besonderheiten von Node.js wird abgeleitet, für welche Aufgabenstellungen es überhaupt geeignet ist, um anschließend anhand einer Vielzahl von praktischen Beispielen die Verwendung relevanter Konzepte und Bibliotheken zu zeigen. Besonderes Augenmerk gilt auch Bereichen wie Security, Qualitätssicherung und Betrieb.
